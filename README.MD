# Order of production

>>> Open the app with http://localhost:8000/api/v1/to-do

1. Define routes.

2. Define the verbs of each route.
>>> Example:

/todo
- GET (Get all "to do's")
- POST (Create a new "to do")

/todo/:id
- GET (Get "to do" by ID)
- PUT (Update any "to do")
- DELETE (Delete any "to do")

3. Create controllers.

4. Create services.

5. Create routes.
